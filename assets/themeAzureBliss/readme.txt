Thank you for purchasing and downloading AzureBliss!
Please contact me through WrapBootstrap if you have any questions or comments.

This theme includes LESS and CSS files. By default, the CSS files are enabled. To use LESS instead of CSS, uncomment the LESS Stylesheets in each HTML directory and comment out the CSS ones.
To edit the LESS files, just simply edit app.less in the less directory. Compile app.less with WinLess or another compiler when you are done.
